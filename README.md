# Jogo da Velha - Projeto Personalizado

Este é um **Jogo da Velha** desenvolvido com HTML, CSS e JavaScript. O objetivo do projeto foi criar uma versão funcional do clássico jogo, com a adição de algumas funcionalidades extras e estilizações para melhorar a experiência do usuário.

## Funcionalidades

- **Jogo funcional de Jogo da Velha**: Implementação completa do jogo de 2 jogadores, alternando entre os jogadores X e O.
- **Contador de Jogos**: Ao final de cada partida, o contador de jogos é incrementado. Também há um contador para as vitórias de cada jogador (X e O).
- **Alteração de cores**: Os quadrados do jogo mudam de cor quando um vencedor é encontrado, destacando a linha/coluna/diagonal vencedora.
- **Animação**: O jogo possui animações nos quadrados ao passar o mouse sobre eles e ao clicar, oferecendo uma interação mais dinâmica.
- **Estilo moderno e personalizável**: O visual do jogo foi aprimorado com cores vibrantes e interatividade com o usuário, além de uma opção de alternância de tema (claro/escuro).
- **Exibição de vencedor**: Ao final de cada partida, o vencedor é exibido na tela com destaque.

## Funcionalidades Adicionais

- **Contagem de vitórias de X e O**: Cada vitória de um jogador é contada e exibida em tempo real no painel.
- **Botão de reinício**: Após cada vitória, o botão de reinício permite que os jogadores comecem uma nova partida sem perder as estatísticas.
  
## Tecnologias Usadas

- **HTML**: Para estruturar o conteúdo da página.
- **CSS**: Para estilizar a interface do jogo, tornando-a mais atraente e interativa.
- **JavaScript**: Para implementar a lógica do jogo, a alternância de jogadores, verificação do vencedor e o contador de vitórias.

## Modificações Realizadas

O esqueleto básico do jogo foi ensinado pelo professor, e eu fiz diversas alterações para personalizá-lo e melhorá-lo:

1. **Animações nos quadrados**: Adicionei animações de hover e click para tornar a interação mais fluída e divertida.
2. **Contador de Jogos e Vitórias**: Implementei contadores para registrar o número de jogos jogados, bem como a quantidade de vitórias de cada jogador (X e O).
3. **Alteração no Estilo**: Melhorei o estilo do jogo com uma paleta de cores mais atraente, incluindo efeitos visuais para os quadrados vencedores.
4. **Feedback visual e sonoro**: Alterei o layout para mostrar o vencedor de forma destacada após cada jogo e usei animações para tornar o jogo mais envolvente.
5. **Reinício do Jogo**: A implementação do botão "Reiniciar" permite que o jogo reinicie facilmente sem perder as estatísticas.

## Como Usar

1. Clone este repositório para sua máquina local: https://github.com/sllnoliveira/jogo-da-velha1.git

2. Abra o arquivo `index.html` em seu navegador para jogar.
3. As estatísticas e a pontuação serão atualizadas automaticamente após cada jogo.
4. Para reiniciar o jogo, clique no botão "Reiniciar".
